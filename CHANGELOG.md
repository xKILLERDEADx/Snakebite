# Changelog

All notable changes to Snakebite will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [2.0.0] - 2026-02-28

### Added
- 219+ security scanning modules covering recon, injection, OSINT, and more
- Advanced SQLi scanner (Error-Based, Boolean-Based, Time-Based, UNION-Based)
- Context-aware XSS detection with DOM-based analysis
- Full OWASP Top 10 compliance assessment
- CVE mapping and exploit intelligence engine
- AI-powered vulnerability classification
- Live HTML dashboard with real-time findings
- PDF and HTML report generation
- Nuclei template engine integration
- Dark web & breach monitoring
- Cloud metadata SSRF detection
- OAuth/SAML security analysis
- DNS zone transfer and subdomain enumeration
- WebSocket, GraphQL, and HTTP/2 protocol scanners
- Rate limiting, WAF detection, and bypass modules
- Plugin system for custom extensions
- Telegram & Discord scan notification support
- Scan checkpointing and resume capability
- Proxy chain support for anonymized scanning
- Multi-target scanning from file input

### Security
- Legal disclaimer and responsible use warnings
- Rate limiting to prevent target overload

## [1.0.0] - 2025-01-01

### Added
- Initial release with core scanning modules
- Basic reconnaissance and vulnerability detection
