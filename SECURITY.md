# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 2.x     | :white_check_mark: |
| < 2.0   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within Snakebite, please report it responsibly.

**Do NOT open a public issue.**

Instead, please email us at: **spaceworkofficial@gmail.com**

We will acknowledge your email within 48 hours and work with you to understand and resolve the issue promptly.

## Responsible Use

Snakebite is designed for **authorized security testing only**. Usage of this tool against targets without prior mutual consent is illegal. Users are responsible for complying with all applicable laws and regulations.
